Utiliser la ligne de commande

Tout au long de ce livre, on trouve de nombreux exemples d'exécution de programmes Python depuis la ligne de commande. Mais où se trouve cette ligne de commande ?

Sur Linux, cherchez dans votre menu Programmes un programme appelé Terminal. (Il pourrait se trouver dans un sous-menu comme Accessoires ou Système.)

Sur Mac OS X, il existe une application appelée Terminal dans votre répertoire /Applications/Utilities/. Pour y accéder ouvrez le menu Aller du Finder, et sélectionnez Aller au dossier..., finalement tapez /Applications/Utilities/. Il suffit de double-cliquer sur l'icône Terminal pour lancer le programme.

Sur Windows, dans le menu Démarrer, sélectionnez Exécuter..., tapez cmd, et appuyez sur la touche ENTRÉE.

Lancer Python depuis la ligne de commande

Une fois que vous avez lancé la ligne de commande, vous devriez être à même de lancer l’invite de commande interactive de Python. Sur Linux ou Mac OS X, tapez python3 et appuyez sur ENTRÉE. Sur Windows, tapez c:\python31\python et appuyez sur ENTRÉE. Si tout se passe bien, vous devriez voir quelque chose comme ce qui suit: 

you@localhost:~$ python3
Python 3.1 (r31:73572, Jul 28 2009, 06:52:23) 
[GCC 4.2.4 (Ubuntu 4.2.4-1ubuntu4)] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>>

(Tapez exit() et appuyez sur ENTRÉE afin de sortir de l’invite de commande interactive de Python et ainsi récuperer la ligne de commande. Cette méthode fonctionne sur tous les systèmes d'exploitation.)

Si vous obtenez un message d’erreur disant “command not found” cela veut très certainement dire que Python 3 n’est pas installé. 

you@localhost:~$ python3
bash: python3: command not found

Si par contre l’invite de commande de Python s’affiche correctement mais que le numéro de version n'est pas celui que vous attendiez, vous avez certainement plusieurs versions de Python installées sur votre ordinateur. Cela arrive souvent sur Linux et Mac OS X, où l’on peut trouver de vieilles versions de Python pré-installées. Vous pouvez installer la toute dernière version de Python sans supprimer les autres versions, en effet plusieurs versions peuvent co-habiter sur un même système. Par contre vous devrez être plus spécifique quand à la version de Python que vous désirez lancer depuis la ligne de commande.

Par exemple, sur mon système Linux, j’ai plusieurs versions de Python installées afin de pouvoir tester les programmes que j’écris. Pour lancer une version spécifique je peux saisir au clavier python3.0, python3.1, ou python2.6. 

mark@atlantis:~$ python3.0
Python 3.0.1+ (r301:69556, Apr 15 2009, 17:25:52)
[GCC 4.3.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> exit()
mark@atlantis:~$ python3.1
Python 3.1 (r31:73572, Jul 28 2009, 06:52:23) 
[GCC 4.2.4 (Ubuntu 4.2.4-1ubuntu4)] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> exit()
mark@atlantis:~$ python2.6
Python 2.6.5 (r265:79063, Apr 16 2010, 13:57:41) 
[GCC 4.4.3] on linux2
Type "help", "copyright", "credits" or "license" for more information.
>>> exit()
