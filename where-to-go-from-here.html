Les choses à lire

Malheureusement je ne peux pas couvrir toutes les facettes de Python 3 en détail dans ce livre. Mais heureusement il existe d'excellents tutoriels sur internet qui le feront à ma place.

Les décorateurs:

  • Les décorateurs de fonctions par Ariel Ortiz
  • Un peu plus sur les décorateurs de fonctions par Ariel Ortiz
  • Charming Python : Les décorateurs facilitent la magie par David Mertz
  • Les définitions de fonctions dans la documentation officielle de Python

Les propriétés:

  • Le module property intégré à Python par Adam Gomaa
  • Getters/Setters/Fuxors par Ryan Tomayko
  • La fonction property() dans la documentation officielle de Python

Les descripteurs:

  • Manuel des descripteurs par Raymond Hettinger
  • Charming Python : De l'élégance et des défauts de Python, Partie 2 par David Mertz
  • Les descripteurs Python par Mark Summerfield
  • Utilisation des descripteurs dans la documentation officielle de Python

Threading & multi-processing:

  • Le module threading de Python
  • threading — Gérer des threads concurrents
  • Le module multiprocessing
  • multiprocessing — Gérer les processus comme des threads
  • Les threads Python et le Global Interpreter Lock par Jesse Noller
  • A l'intérieur du Python GIL (video) par David Beazley

Les méta-classes:

  • L'utilisation des méta-classes en Python par David Mertz and Michele Simionato
  • L'utilisation des méta-classes en Python, Partie 2 par David Mertz and Michele Simionato
  • L'utilisation des méta-classes en Python, Partie 3 par David Mertz and Michele Simionato

En supplément, Le Module Python de la semaine de Doug Hellman est un très bon guide aux différents modules de la librairie standard de Python.
